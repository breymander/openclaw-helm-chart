OpenClaw Helm Chart Repository

Repository URL: https://breymander.github.io/openclaw-helm-chart

This is a Helm chart repository for deploying OpenClaw, an open-source personal AI assistant powered by Anthropic's Claude LLM.

Installation

Add this repository to Helm:

helm repo add openclaw https://breymander.github.io/openclaw-helm-chart
helm repo update

Install the chart:

helm install my-openclaw openclaw/openclaw
